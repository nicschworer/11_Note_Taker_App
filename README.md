  [![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

## Description 
This Note Taker app can be used to write, review and manage user input notes. Using express.js the application uses a series of get, post, and delete requests to read and write to a json file which houses your saved notes.  

## Credits
This package uses node.js and express.
